Assignment #5: Creating a Dynamic Table

This assignment consists of two parts. The first part was a user form which accepts four numbers from the user, validates them and lets the user know what needs to be corrected if need be. Once the form is submitted it processed on another page using JavaScript. The processing takes the user input using the GET method and creates a multiplication table with CSS styling.
